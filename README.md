# Formula 1 2015: Hamilton Dominance & Season Performance Analysis

**OVERVIEW**
This project analyzes the 2015 formula 1 season using Power BI, Focusing on championship dominance, driver consistency, podium finishes, and constructor efficiency.
The Dashboard highlights Lewis Hamilton's commanding season while also exploring broader competitive trends across teams and drivers.

**OBJECTIVES**
- Evaluate championship performance across all drivers
- Identify the most consistent and successful drivers
- Analyze podium trends across the season
- Compare constructor efficiency and reliability
- Measure how starting grid position influenced final race outcomes

**KEY INSIGHTS**
- Lewis Hamilton dominated multiple performance categories including wins, podiums, and points leadership
- Teams with stronger reliability maintained competitive advantage throughout the season'
- Drivers gaining positions consistently demonstrated stronger race pace than qualifying pace.
- Starting near the front significantly improved finishing outcomes

**TOOLS USED**
- Power BI
- Power Query
- DAX
